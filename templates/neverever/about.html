about us

A bit of information on the project and its developers.

the project

This web application was built as the assessed project for the Master's level course Internet Technology 2014/15. You can find all our code in our GitHub repository. never/ever was built using the Django framework with the aid of Tango with Django by Dr. Leif Azzopardi. The live version of this project is hosted on PythonAnywhere and was visually enhanced by using Bootstrap and the (unofficial) Bootstrap Switch.

the team

We, Team N00bel Prize, are all postgraduate students of the University of Glasgow, taking the course Internet Technology. With one Computing Science student, who as previous programming experience, and three Information Technology students, who have no previous experience; we have a diverse team with and have yet to learn how to develop professional web applications. Still, we hope you can enjoy our work and please feel encouraged to contact us in case you have any questions.
